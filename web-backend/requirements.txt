# ============================================================================
# 版本兼容性说明
# ============================================================================
#
# Python版本要求: 3.9+
# 原因: 
# - pydantic-core 2.x 需要Python 3.9+
# - 使用了现代Python特性如类型注解、dataclasses等
#
# 已知兼容性问题:
# - Python 3.13: pydantic-core可能编译失败，建议使用Python 3.9-3.11
# - 某些系统可能需要安装额外的系统依赖（如gcc、python3-dev等）
#
# 安装建议:
# 1. 使用conda创建隔离环境: conda create -n penrecon python=3.9
# 2. 激活环境: conda activate penrecon
# 3. 升级pip: pip install --upgrade pip
# 4. 安装依赖: pip install -r requirements.txt
#
# 如果遇到编译错误:
# - 确保安装了系统开发工具: sudo apt install build-essential python3-dev
# - 尝试使用预编译的wheel包: pip install --only-binary=all -r requirements.txt
